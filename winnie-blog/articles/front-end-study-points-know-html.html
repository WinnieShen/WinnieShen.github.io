1.Hello World-编写你的第一个html页面

第一步，新建一个记事本，随意命名，最好是英文的，注意，此时后缀是.txt。
第二步，复制下面的代码到刚刚新建的记事本当中，保存。
<html>
<body>
    <p>Hello World!</p>
</body>
</html>
第三步，修改后缀.txt为.html。
第四步，用任意浏览器（我这里用的是Google的Chrome）打开刚才所建的文件，就可以看到"Hello World!"这几个字母。

2.什么是html-html相关的概念

这里我直接引用w3school中的知识，来解释，当然了， w3school是一个非常不错的官方教程。
HTML：
  • HTML 是用来描述网页的一种语言。
  • HTML 指的是超文本标记语言 (Hyper Text Markup Language)
  • HTML 不是一种编程语言，而是一种标记语言 (markup language)
  • 标记语言是一套标记标签 (markup tag)
  • HTML 使用标记标签来描述网页
HTML标签：
  • HTML 标记标签通常被称为 HTML 标签 (HTML tag)。
  • HTML 标签是由尖括号包围的关键词，比如<html>
  • HTML 标签通常是成对出现的，比如<p>和</p>
  • 标签对中的第一个标签是开始标签，第二个标签是结束标签
  • 开始和结束标签也被称为开放标签和闭合标签
HTML 文档 = 网页
  • HTML 文档描述网页
  • HTML 文档包含 HTML 标签和纯文本
  • HTML 文档也被称为网页
<html>
<body>
    <p>Hello World!</p>
</body>
</html>
Hello World中的标签解释
  • <html>与</html>之间的文本描述网页
  • <body>与</body>之间的文本是可见的页面内容
  • <p>与</p>之间的文本被显示为段落
