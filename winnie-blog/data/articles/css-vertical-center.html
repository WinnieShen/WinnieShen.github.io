在前端技术日新月异的今天，浏览器兼容已经成为一个重要问题，那么CSS中实现的样式，也需要考虑到浏览器的兼容，那么，我们今天来探讨一下垂直居中的方式，以及在不同浏览器的实现情况。

1.设置 display:table

<div id="wrapper">
    <div id="cell">
        <div class="content">
            Content goes here!
        </div>
    </div>
</div>

#wrapper {
    width: 400px;
    height: 200px;
    background-color: orange;
    display: table;
}

#cell {
    display: table-cell;
    vertical-align: middle;
}
支持浏览器:Firefox,chrome,IE8+,Safari等，但是标签太过繁杂，在content的高度不确定的情况下，可以使用。
点击查看运行效果

2.设置 position:absoulte

<div class="content"></div>

.content {
    background-color: orange;
    height: 240px;
    width: 100px;
    position: absolute;
    top: 50%;
    margin-top: -120px;
}
支持浏览器:基本适用于所有的浏览器，在content的高度已知的情况下，可以使用。
点击查看运行效果

3.line-height

<div class="content">
     Content goes here!
</div>

.content {
    background-color: orange;
    height: 100px;
    line-height: 100px;
}
支持浏览器:基本适用于所有的浏览器，这种方法也是在content的高度已知的情况下使用，和第2种方法不同的是，这是设置content内部的内容垂直居中，而内容仅限于文字， 需要注意的是，当内容多于1行时，内容便会溢出。
点击查看运行效果

4.使用margin:auto

<div class="content"></div>

.content {
    background-color: orange;
    position: absolute;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
    margin: auto;
    height: 240px;
    width: 70%;
}
支持浏览器:Firefox,chrome,IE8+,Safari等，content会随窗口的大小而变化，始终位于屏幕中央。
点击查看运行效果

5.在content上面加个div，高度占据整个页面的50%,并且进行位置的调整

<div id="wrapper"></div>
<div id="content"></div>

html,body {
    height: 100%;
    margin: 0;
    padding: 0;
}

#content {
    height: 50%;
    margin-bottom: -150px;
}

#content {
    height: 300px;
    background-color: orange;
}
支持浏览器:基本适用于所有的浏览器。wrapper的margin-bottom为-(content的height)/2。
点击查看运行效果
